== Basic TicTacToe

This rails 4.1 app is a basic implementation of the TicTacToe game.
* Click the New Game link in the menu bar
* Enter the player's name in the name field
* Enter the name of the game in the game field
* The second player must use the same game name in order to participate in that game
* Use two separate browsers when playing a game locally. This is necessary because browsers (even separate windows) will maintain the session
